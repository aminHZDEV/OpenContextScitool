# Example auth-service bundle

A tiny hand-written OKF bundle for the docs of a fictional auth service. It shows
the concept format plus the two features that distinguish this tool: a `Caveat`
(a documented trap) and a `conflicts_with` pair (two docs that disagree).

⚠ The token TTL is documented inconsistently — see [the conflict](./caveat-token-ttl-conflict.md).
